## 🚀 Website Live Karne ke Steps

### Step 1: GitHub pe Upload karo

1. **github.com** par jaao aur free account banao
2. Green **"New"** button click karo
3. Repository name: `bitmillion-website`
4. **"Create repository"** click karo
5. **"uploading an existing file"** click karo
6. Is folder ki **saari files** drag & drop karo
7. **"Commit changes"** click karo

### Step 2: Vercel pe Deploy karo

1. **vercel.com** par jaao
2. **"Sign up with GitHub"** click karo
3. **"Add New Project"** click karo
4. `bitmillion-website` select karo
5. **"Deploy"** click karo
6. 2-3 minute wait karo ✅

### Step 3: Website Ready!

Aapko milega: `bitmillion-website.vercel.app`
